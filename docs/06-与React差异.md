# 与完整 React 的差异

本文列出当前实现的简化点与扩展提示。文档索引见 **[AGENTS.md](../AGENTS.md)**。

## 预期内的简化

以下有助于阅读源码时建立正确预期：

- **调度**：无并发特性；**同步**执行整棵树的 render + commit。
- **特性**：仅实现部分 API 与节点类型（以仓库内 `beginWork` / `completeWork` 分支为准）。
- **属性更新**：Host 组件属性在注释中仍有与 `Update` flag、commit 路径对齐的预留空间。

## 扩展建议

若后续为项目补充单元测试或自定义 Test Host，可与 **hostConfig** 解耦设计一致，在宿主边界外增加可测层。
