# 2026-05-04

## 发现

- 初始化 memory 日期归档记录。
- Vite 在 `packages/demos/src/main.js` 中解析到 JSX 时报错；根因是 `.js` 默认按普通 JS 处理，不进行 JSX import analysis。
- 将 demos 入口调整为 `src/main.jsx` 并更新 `index.html` 的模块引用后，问题消失，符合 Vite 默认约定。

## 决策

- demos 入口文件采用 `.jsx` 命名，而不是在 Vite 中放宽 `.js` JSX 解析规则。
- 优先遵循工具默认行为，降低后续维护成本与配置歧义。
