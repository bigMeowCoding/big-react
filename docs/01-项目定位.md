# 项目定位

本文说明 **big-react** 的学习目标、技术栈与运行环境。完整文档索引见仓库根目录 **[AGENTS.md](../AGENTS.md)**。

**big-react** 是一个用于学习的 **React 核心原理实现**：在体量上做了简化，但保留了与真实 React 相近的 **Fiber 双缓冲、Render / Commit 两阶段、副作用标记（flags）、子节点协调（reconcile）** 等主干结构。

## 技术栈与运行环境

- **构建**：pnpm workspace；包构建用 Rollup；演示应用用 Vite（`pnpm dev`）。
- **运行时**：浏览器环境；`react-dom` 通过 `document` 创建与操作 DOM。
