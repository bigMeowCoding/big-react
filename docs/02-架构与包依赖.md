# Monorepo 架构与包依赖

本文说明各包职责与协作关系。文档索引见 **[AGENTS.md](../AGENTS.md)**。

## 目录结构

```
packages/
├── react/              # 对外 API：useState、内部 Dispatcher 引用
├── react-reconciler/   # Fiber 树、调度（当前为同步）、协调与提交
├── react-dom/          # DOM Host：createRoot、hostConfig、合成事件入口
├── shared/             # ReactSymbols、react ↔ reconciler 的 internals 桥接
└── demos/              # 示例应用（Vite）
```

## 依赖关系（概念上）

- **`react`**：提供 `useState`；通过 `__SECRET_INTERNALS_DO_NOT_USE_OR_YOU_WILL_BE_FIRED` 暴露 `currentDispatcher`。
- **`shared/internals`**：从 `react` 再导出 internals，供 `react-reconciler` 在 `renderWithHooks` 里**注入** Hooks 调度实现（mount/update 两套 dispatcher）。
- **`react-reconciler`**：实现 `createContainer` / `updateContainer`、工作循环与提交；**宿主操作**通过引用 `react-dom/hostConfig`（如 `createInstance`、`appendChild`）完成，从而与具体环境（此处为 DOM）解耦。
- **`react-dom`**：挂载根容器、初始化事件、调用 `updateContainer`。

更细的设计（Fiber、flags 等）见 [03-核心概念](./03-核心概念.md)。
